## Run project in local environment

```bash
#도커 사용시(로컬 DB 따로 구성하지 않아도 됨.)
$ docker compose up --build

# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```
